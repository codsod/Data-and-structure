# 1.题目
### 斐波那契数列
思路：在递归的基础上，运用动态规划，减少计算复杂度，防止超时
#### 代码
```
def f(n):
    if n<=2:
        return 1
    if dp[n]!=-1:
        return dp[n];
    else:
        dp[n]=f(n-1)+f(n-2)
        return dp[n]

dp=[-1]*21
n=int(input())
ans=[]
for _ in range(n):
    num=int(input())
    ans.append(str(f(num)))

print('\n'.join(ans))
```
